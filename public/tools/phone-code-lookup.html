What is Phone Code Timezone Lookup?

When you see an international phone number like +91 98765 43210 or +44 20 7946 0958, wouldn't it be helpful to instantly know what timezone that person is in? That's exactly what this tool does.

Simply enter the country dial code (the part after the +), and you'll immediately see:

  • The country associated with that phone code
  • All timezones used in that country
  • Current local time in each timezone
  • GMT/UTC offset for each timezone
