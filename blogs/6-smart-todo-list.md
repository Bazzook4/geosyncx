## How to Use Smart To-Do

### Step 1: Open the Task Tab

1. Go to [GeoSyncX](https://www.geosyncx.com)
2. Click the **"Smart To-Do"** tab
3. You'll see the task input and your existing tasks (if any)

### Step 2: Add Your First Task

1. Click in the input field
2. Type a short task description
3. Hit Enter or click "Add"

The task appears at the top of your list.

### Step 3: Add More Tasks

Repeat the process. Tasks stack in the order you add them (newest first).

**Pro tip:** Add tasks as they come up during the day. Don't wait to create a "complete" list.

### Step 4: Check Off Completed Tasks

As you finish tasks:
1. Click the checkbox next to the task
2. It gets strikethrough styling
3. But stays visible (so you remember what you accomplished)

**Why not auto-hide?** Seeing completed tasks is motivating. You can see your progress for the day.

### Step 5: Share Your Tasks

**For one task:**
- Click "Copy Bullet" next to the task
- Paste in Slack/email/docs

**For all tasks:**
- Click "Copy Bullets" button at the top
- Paste the entire list

Common use cases:
- Daily standup updates: "Yesterday I: [paste bullets]"
- Status reports: "Completed this week: [paste bullets]"
- Delegation: "Can you handle: [paste 3 specific bullets]"

### Step 6: Clean Up

Periodically delete tasks you no longer need:
- Old completed tasks
- Cancelled tasks
- Tasks moved to another system

Keeps your list focused on current work.

## Pro Tips

### Tip 1: Keep Tasks Short

Good task length: **2-5 words**
- âœ“ Email Julia
- âœ“ File taxes
- âœ“ Update proposal

Bad task length: **20+ words**
- âœ— Email Julia about the Q3 marketing proposal and ask if she needs the budget spreadsheet updated before Friday's meeting

Why? Short tasks are:
- Easier to scan
- More actionable
- Better for sharing

Use the details section of actual project management tools for long descriptions.

### Tip 2: Use Action Verbs

Start each task with a verb:
- Email... Call... Send... Review... Update... Fix... Write...

Verbs make tasks feel actionable, not just notes.

### Tip 3: One Task = One Action

Bad: "Handle all Client A tasks"
Good:
- Send proposal to Client A
- Schedule call with Client A
- Invoice Client A for deposit

Why? You can check off progress incrementally.

### Tip 4: Add Tasks Immediately

Don't wait to "organize" tasks. When something comes up:
1. Add it immediately
2. Check it off when done

Capture > Perfect organization.

### Tip 5: Daily Reset

At the end of each day:
1. Delete all completed tasks
2. Start tomorrow fresh

Or keep them for a weekly review, then bulk delete.

Find the rhythm that works for you.

### Tip 6: Combine with Calendar

**To-Do vs Calendar:**
- **Calendar:** Time-specific commitments (meetings, deadlines)
- **To-Do:** Flexible tasks (do anytime today)

Use both:
- Schedule meetings in Google Calendar via GeoSyncX

- Perfect combo

### Tip 7: Use for Meeting Agendas

Instant meeting minutes.

## Common Questions

**Q: How many tasks can I add?**
A: No limit (within browser storage constraints, which is typically several thousand tasks). But we recommend keeping it under 20 for daily focus.

**Q: Can I reorder tasks?**
A: Currently tasks appear in the order added (newest first). Reordering isn't available, but you can delete and re-add to change order.

**Q: Can I edit a task after adding?**
A: Currently, no. Delete and re-add if you need to change it. We keep the UI simple intentionally.

**Q: What if I clear my browser data?**
A: All tasks are deleted. Export important tasks (copy bullets, paste into a doc) before clearing browser data.

**Q: Can I sync across devices?**
A: No cloud sync. Each browser/device has its own task list. This ensures privacy but means tasks don't sync.

**Q: Can I add due dates or reminders?**
A: No. Smart To-Do is intentionally simple. Use Google Calendar or dedicated task apps for time-sensitive tasks.

**Q: Can I assign tasks to others?**
A: No direct assignment, but you can copy bullets and paste into messages to delegate.

**Q: Is there a mobile app?**
A: GeoSyncX is web-based and works on mobile browsers. Open www.geosyncx.com on your phone.
